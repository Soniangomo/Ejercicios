World Wide Web

En informatica, la World Wide Web (WWW) o Red informatica mundial comunmente conocid como la web, es un sistema de disribucion de documentos de hipertexto o hipermedios interconectados y accesibles via Internet. Con un navegador web,un usuario visualiza sitios web compuestos de paginas web que pueden contener texto, imagenes,videos u otros contenidos multimedia, y navega a traves de esas paginas usando hiperenlaces
